### Version 1.4.4 Release Notes
* **New Welcome Experience**: Introduced a dedicated onboarding/upboarding page to guide users through new features and privacy settings.
* **Customizable Toolbar Actions**: Configure the toolbar icon for a Dropdown menu (default), immediate Search, or immediate Archive via Options.
* **Fixed Extension Removal**: Restored the "Remove" button functionality in both Options and Welcome pages to correctly uninstall the extension.
* **Streamlined Context Menus**: Removed redundant Search context menu item (except for immediate Archive) for a cleaner right-click interface.
* **Auto-Save Options**: Options now save immediately upon change with visual confirmation; the "Save" button is now a "Close" button.
