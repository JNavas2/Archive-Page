Pin extension icon to your browser toolbar for easy access.

What's New in 1.4.4

  • Dropdown menu (Search, Archive, Options) (default)
  • Search (immediate action)
  • Archive (immediate action)

Important Notes

  1. Search context menu now only in Archive (only) option.
  2. To restore prior behavior, select Archive (only) in Options.

Privacy

By using this extension you consent to sharing your Archive or Search URL with Archive Today when and only when initiated by you. If you do not consent, Remove this extension, which does not otherwise collect or share any user data.
